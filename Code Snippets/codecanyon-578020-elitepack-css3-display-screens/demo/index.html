Tablet

Tablets


Elitepack CSS3 tablets are a nice modern way of showcasing your application, website or portfolio gallery using pure CSS3 powered models. The tablet pack features some nice graphical elements to showcase your application, website or portfolio on a tablet.

Let's start off with a plain tablet

Here is a breakdown of the classes used to create the tablet:

  • 'tablet' class defines the main frame of the tablet container
  • 'screen-t' class defines the screen element
  • 'addons-t' class defines the camera and power/volume buttons
  • 'button-t' class defines the main button

optional classes:

  • 'glare-t or glow-t' class defines a light glare or glow effect onto the tablet
  • 'rotate' class rotates the tablet 90 degrees clockwise
  • 'stack' class can be used to stack couple of tablets together
  • 'white' class changes the color to a white one

  • View HTML Code

    <div class="tablet" >
	<div class="screen-t"></div>
	<div class="addons-t"></div>
	<div class="button-t"></div>
</div>

You can add some lighting effect: Glare or glow

By adding a div with the glare-t or glow-t class, it will add a lighting effect onto the screen of the tablet

Cool! How about a different orientation?


By using the rotate class within the tablet div, it rotates the tablet with an angle of 90 degrees clockwise. When using the rotate class, use a background that is wider than long i.e 300 by 240 px to ensure the quality of the image remains as is.

  • View HTML Code

    <div class="tablet rotate">
<div class="glare-t"></div>
<div class="screen-t"></div>
<div class="addons-t"></div>
<div class="button-t"></div>
</div>

Let's add an image to the screen

To add an image simply add the <img> tag within the div with the screen-t class. By default, the dimensions of the image has been set to 240px by 300px. You'll notice if you use a bigger image it will be resized to fit to these dimensions. However, this can be changed in the css stylesheet

  • View HTML Code

    <div class="tablet">
	<div class="glare-t"></div>
	<div class="screen-t">
		<img src="images/tablet.png" alt="tablet" />  
	</div>
	<div class="addons-t"></div>
	<div class="button-t"></div>
</div>

You can use any other kind of media like youtube or vimeo videos

Great! Now, let's stack a couple of them together


To stack couple of tablets together, add the stack class within the tablet div. Take note, the first tablet should NOT have a stack class.

I have used a little 'depth' trick by adding an opacity class to the images on the stacked backgrounds. Notice how the top tablet has a more sharper image than the last one on the stack. You do so by adding, for example, a vis75 class to the image to make its opacity be at 0.75

Can I put a shadow effect on them? Sure!

Shadows are a nice way to even further add '3D effects' to your tablet. you can add a shadow by adding the top-light-shadow class within the tablet class. For stacked tablets, you'll notice they have a left shadow which is important to define depth in stacked items

  • View HTML Code

    <div class="tablet top-light-shadow">
<div class="glow-t"></div> 
<div class="screen-t"></div>  
<div class="addons-t"></div>
<div class="button-t"></div>  
</div>
